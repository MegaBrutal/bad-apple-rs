# Run

```bash
git clone git@github.com:lomirus/bad-apple-rs.git
cd bad-apple-rs
cargo run
```

# Screen Requirements

- WIDHT >= 101 Chars
- HEIGHT >= 44 Chars

Succeed in Windows Terminal full screen mode with default configurations in Windows 10, 1920x1080 Resolution, 125% Scale.
